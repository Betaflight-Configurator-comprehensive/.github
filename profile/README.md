## What is Betaflight Configurator

**Betaflight Configurator** is an end-to-end desktop utility for Betaflight flight controllers. It streamlines USB/UART connection, target selection and flashing, initial calibration, UART/RX configuration (SBUS/CRSF/ELRS), VTX/OSD setup, PID and filter tuning, profile management, backups and Blackbox log workflows. Built for freestyle, racing and long-range builds.

## Getting Started

1. Install the application on your PC.  
2. Connect the flight controller via USB and select the COM port.  
3. Create a backup, flash the selected target if needed.  
4. Configure UARTs, receiver, VTX/OSD, base parameters and modes.  
5. Tune PID/filters, save profiles, test-fly and iterate.
